## Our Code of Conduct

All working groups at ESIIL are beholden to our Code of Conduct. All team members, PIs, and the collaboration lead is responsible for having read our Code of Conduct prior to arriving at ESIIL. If you have questions about our Code of Conduct please reach out to our Community Care Lead, Susan Sullivan. You can find a link to our Code below.
